A modern and responsive Gym Landing Page built using HTML and CSS.
This project showcases a fitness brand website with pricing plans and plan comparison.

Project Overview

Sahib Fitness is a landing page designed to promote a gym business.
It includes:

Navigation bar

Hero section

Pricing section

Plan comparison table

Footer

This project focuses on layout design using Flexbox and reusable utility classes.

 Features

Responsive layout using Flexbox

Clean and modern UI

Pricing cards (Basic, Pro, Premium)

Comparison table for plans

Styled buttons with hover effects

Utility CSS classes for reusable styling

Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout

Google Fonts (Poppins) – Typography

Flexbox – Layout system
